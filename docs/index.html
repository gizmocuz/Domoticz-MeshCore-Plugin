MeshCore Dashboard

A real-time, in-browser dashboard for your MeshCore LoRa mesh — node telemetry, messaging, topology, history and analytics served straight from the Domoticz plugin over a native WebSocket channel. This page is a static feature tour; install the plugin to get the interactive version.

Install View on GitHub Dashboard docs

Feature tour

🗺️ Mesh topology & path lines

Leaflet map plots every contact (and optionally every heard node) and draws the real multi-hop repeater paths recent messages traversed.

📈 Historical analytics

Time-series side panel — RSSI, SNR, noise floor, RX/TX packet volume, hourly traffic, top relay keys and per-channel message stats. SQLite-backed, schema-versioned, 30-day window.

👂 Heard nodes

Persistent list of mesh nodes that aren't in your contacts. Per-node advert count, per-type filters, GPS pin, and bulk purge by age or low hit-count.

🚿 RX firehose

Live packet stream with type filters, pause/resume, and per-row SNR/RSSI/path-len breakdown for deep mesh debugging.

📊 Statistics

Lifetime totals and leaderboards — messages, adverts, top senders, deepest hops, longest-running relay keys.

🍩 Packet types

Donut breakdown of RX payload types — adverts, channel messages, DMs, ACKs, status responses, anonymous requests…

Other highlights

Native WebSocket transport. No JSON-file or JSON-API polling — every update is pushed by the plugin in real time.
SQLite message store. Single source of truth for the inbox and per-contact DM conversations, with server-side pagination and full-text search.
Compose & reply. Send to channels or contacts, emoji picker, @mention highlighting, hover-to-reply on any message.
Self-node telemetry. Battery, voltage, RSSI, SNR, noise floor, uptime, airtime and packet counters — exposed as Domoticz devices.
Auto-discovery. Every contact advertised by the mesh is tracked automatically — no manual node list.
Delivery ACK markers. Outgoing DMs show a delivered tick when the recipient acknowledges, even multi-hop.
Coverage & line-of-sight tools. Terrain-aware propagation map and per-link path-profile chart with Bullington knife-edge attenuation.
Activity heatmap. 24-hour rolling Leaflet heatmap of RX events on the topology map.
dzVents integration. Bundled status-report and chat-responder example scripts — automations driven by mesh messages.
Auto-reconnect & stale detection. Worker thread reconnects after connection loss; stale-link watchdog triggers a reconnect after 10 min without push events.
